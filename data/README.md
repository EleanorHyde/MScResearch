# RAW and Filtered Data
This folder contains the following raw and filtered data:
- Questionnaire_Data_Download_22.04.2021 : the raw questionnaire data downloaded from Qualtrics
- Tatool_Data_Downloaded_21.04.2021 : the raw tatool data for each participant downloaded from Tatool
- all_data_combined_filtered : merge of qualtrics and tatool data on one csv which has then been filtered (see below)

## Filtering
- Removed incomplete tatool datasets
- Removed datasets with an accuracy score under 75%
- Trimmed RTs
